# Instruktioner

1. Klona eller ladda ned projektet till din dator.
2. Öppna mappen `client-side-rendering` i din kodeditor.
3. Öppna `index.html` i en webbläsare.
4. Klicka på knappen "Get data" för att hämta och visa planetinformationen.


### Uppgifter
Följ kommentarerna numrerade 1-4 i `client.js` och genomför uppgifterna.
